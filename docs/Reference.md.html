**Živa Reference** This document exposes the API available through Živa. !!! Note Optional arguments are surrounded by `[]` # System methods Methods to setup, run the live coding enviroment and get info from its current state. Address | Args | Description --------|------|------------ `boot` | `actor:s`
- | Delete the actor (remove its instance). # Instrument methods Methods to use with instruments. Address | Args | Description --------|------|------------ `boot` | `[inputChannels, outputChannels,
server, numBuffers, memSize, maxNodes]` | Boot the server with optional parameteres ## Synths Methods exclusive of synths. !!! Note If the method does not exist, it will be passed on as synth argument. Address | Args | Description --------|------|------------ `deg` | `num|pattern` | Play the given `degree` note in the current scale. `oct` | `num|pattern` | Play in the given `octave`. ## Samples Methods exclusive of samples. Address | Args | Description --------|------|------------ `speed` | `number|pattern` | Play the sample at the given `speed`. Affects the pitch. ## MIDI Methods exclusive of midi. Address | Args | Description --------|------|------------ `speed` | `number|pattern` | Play the sample at the given `speed`. Affects the pitch.
