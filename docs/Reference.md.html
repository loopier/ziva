**Živa Reference** This document exposes the API available through Živa. !!! Note Optional arguments are surrounded by `[]` # System methods Methods to setup, run the live coding enviroment and get info from its current state. Address | Args | Description --------|------|------------ `boot` | `[inputChannels, outputChannels,`
`server, numBuffers, `
`memSize, maxNodes]` | Boot the server with optional parameteres `synths` | | List available synths `samples` | | List available synths # Instrument methods Methods to use with instruments. Address | Args | Description --------|------|------------ `pan` | `num|pattern` | Left-right stereo panning effect. `-1` for left, `1` for right, `0` for center. `pingpong` | | Alternate left and right panning. `randpan` | | Set stereo panning to a random value between `-1` and `1` on every event. ## Synths Methods exclusive of synths. !!! Note If the method does not exist, it will be passed on as synth argument. Address | Args | Description --------|------|------------ `deg` | `num|pattern` | Play the given **degree** note in the current scale. `oct` | `num|pattern` | Play in the given **octave**. `note` | `num|pattern` | Play chromatic note. `0` is root at the current octave. `midinote` | `num|pattern` | Play MIDI note. ## Samples Methods exclusive of samples. Address | Args | Description --------|------|------------ `speed` | `number|pattern` | Play the sample at the given `speed`. Affects the pitch. ## MIDI Methods exclusive of midi. Address | Args | Description --------|------|------------ `speed` | `number|pattern` | Play the sample at the given `speed`. Affects the pitch. # Patterns Apply patterns on lists of values. !!!Note **Syntax:** `[item1, item2, ... itemN].patternName(args)` **Examples:** `[0,2,4].pseq`, `[[0,2,4,7].pseq(3), [0,4].prand(4)].pseq` Address | Args | Description --------|------|------------ `pseq` | `repeats` | Play values of the list in order a number of `repeats`. Defaults to `inf`. `prand` | `repeats` | For a number of `repeats`, take a random value of the list. Defaults to `inf`.
